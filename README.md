# Expense Tracker

A modern and user-friendly expense tracking application that helps users manage their daily finances, monitor spending habits, and stay within budget.

## Features

### Expense Management

* Add new expenses with amount, category, date, and description.
* Edit existing expenses.
* Delete expenses.
* View all recorded transactions.

### Expense Categorization

Track expenses across multiple categories such as:

* Food & Dining
* Transportation
* Shopping
* Entertainment
* Healthcare
* Bills & Utilities
* Other

### Dashboard & Analytics

* Total Expenses Overview
* Monthly Spending Summary
* Category-wise Expense Breakdown
* Visual Charts and Graphs
* Recent Transactions List

### Budget Tracking

* Set monthly budgets.
* Monitor spending against budget limits.
* Receive visual indicators when approaching budget thresholds.

### Search & Filters

* Filter expenses by category.
* Filter expenses by date range.
* Search transactions by description.

### Responsive Design

* Mobile-friendly interface.
* Optimized for tablets and desktops.
* Clean and intuitive user experience.

## Installation

### Clone the Repository

```bash
git clone <repository-url>
```

### Navigate to Project Directory

```bash
cd expense-tracker
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```
